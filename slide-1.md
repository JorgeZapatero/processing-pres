# What is Processing?

- A "sketchbook" for creating interactive, creative software quickly and easily
- Began in 2001 at MIT
  - Java Applets!

- Core ideas:
  - Shareable
  - Approachable to artists / designers / beginner programmers
  - Canvas
  - Draw loop

```javascript
function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
  circle(100, 100, 50)
}
```
