# What is Processing?

- A "sketchbook" for creating interactive, creative software quickly and easily

- Core ideas:
  - Shareable
  - Approachable
  - Canvas
  - Draw loopA
  
- Started at MIT in 2001
  - Java Applets!
  - Reimagined with JavaScript

```javascript
function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
  fill(255,0,0);
  circle(100, 100, 50);
}
```
