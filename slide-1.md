# What is Processing?

- A "sketchbook" for creating interactive, creative software quickly and easily
- Started at MIT in 2001
  - Java Applets!

- Core ideas:
  - Shareable
  - Approachable
  - Canvas
  - Draw loop

```javascript
function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
  fill(255,0,0);
  circle(100, 100, 50);
}
```
